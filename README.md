# Website SMP Negeri 2 Tasikmadu

This is a static website for SMP Negeri 2 Tasikmadu, showcasing the school's profile, achievements, extracurricular activities, and contact information.

## Project Structure

- `web.html`: Main homepage of the website.
- `profil/profil.html`: School profile page.
- `prestasi/prestasi.html`: Achievements page.
- `ekstrakurikuler/ekstrakurikuler.html`: Extracurricular activities page.
- `aseet/`: Directory containing images and assets used in the website.

## How to Run

Since this is a static HTML website, you can run it by opening `web.html` in any modern web browser.

1. Clone or download the repository.
2. Navigate to the project directory.
3. Open `web.html` in your browser.

Alternatively, you can serve it locally using a simple HTTP server:

- Using Python (if installed):
  ```bash
  python -m http.server 8000
  ```
  Then open `http://localhost:8000/web.html` in your browser.

- Using Node.js (if installed):
  ```bash
  npx http-server
  ```
  Then open the provided URL.

## Deployment

This website can be easily deployed to GitHub Pages:

1. Push the code to a GitHub repository.
2. Go to the repository settings.
3. Under "Pages", select the main branch as the source.
4. The website will be available at `https://yourusername.github.io/repository-name/web.html`.
